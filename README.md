# Thực hành 1: Blog Listing Page - NextJS & Tailwind

- **Họ và tên:** Hoang Phu Huy
- **Mã sinh viên:** N23DCCN093

## Mô tả ngắn gọn về bài làm
Dự án được khởi tạo bài bản bằng Next.js (App Router) và Tailwind CSS. 
Các tính năng đã hoàn thiện bao gồm:
1. **Trang chủ (Blog Listing):** Fetch dữ liệu danh sách bài viết từ JSONPlaceholder. Hiển thị dưới dạng Component `BlogCard` tái sử dụng, xếp theo cấu trúc Grid Layout chuẩn Responsive (1 cột mobile, 3 cột desktop).
2. **Components:** Đã tách file rõ ràng cho `Header`, `BlogCard`, và `Badge` hỗ trợ việc tái sử dụng.
3. **Trang chi tiết (Blog Detail):** Sử dụng Dynamic Routing `app/blog/[id]/page.js` bóc tách từng bài viết để render giao diện chi tiết bằng tham số ID.

Dự án có giao diện hiện đại, tối ưu responsive mọi thiết bị và không gặp bất kỳ lỗi biên dịch nào trong quá trình chạy.
*(Đã hoàn thành 100% mục tiêu bài thực hành).*
